# Siemens 300E SQLite3 Call Logger
------

This is a lightweight logging  program that logs calls from a Siemens 300E PBX over a serial connection.

## Requirements
PySerial - http://pyserial.sourceforge.net/

## Usage
Edit call_logger-sqlite.py and makesure that the path to the sqlite DB is where you want it and that the Serial Port and Baud Rate are correct.
